# Datasets — Trilha Tecnológica (CV721A)

Os datasets disponibilizados neste repositório têm **finalidade exclusivamente didática**,
sendo utilizados como apoio às atividades da trilha tecnológica da disciplina
**Fundações (CV721A)**.

## Organização

- `projeto1_fundacoes_rasas/`  
  Dataset utilizado no Projeto 1 — Fundações Rasas.

- `projeto2_fundacoes_profundas/`  
  Dataset utilizado no Projeto 2 — Fundações Profundas.

- `exemplos/`  
  Datasets mínimos para testes e demonstrações.

## Observações Importantes

- Os dados **não representam um caso real específico**.
- Os datasets **não substituem dados de projeto reais**.
- Os resultados obtidos devem ser interpretados de forma crítica.
- O uso dos dados está restrito às atividades acadêmicas da disciplina.

O dimensionamento clássico e o julgamento de engenharia permanecem soberanos.
