### Projeto 1 — Diretrizes Específicas de Machine Learning

- O workflow do Orange será fornecido pela disciplina.
- O uso de regressão linear é obrigatório nesta etapa.
- O objetivo é a compreensão conceitual do modelo, e não a exploração de algoritmos alternativos.
- Alterações no workflow ou uso de modelos adicionais não geram pontuação extra.
