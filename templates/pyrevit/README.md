# pyRevit — CV721A Fundações (Trilha Tecnológica)

Este diretório reúne recursos mínimos e institucionais para apoiar a trilha tecnológica da disciplina **Fundações (CV721A)**.

## Objetivo
Padronizar **parâmetros**, **organização mínima do modelo** e **extração de dados** (export) para conexão com:
- relatórios dos projetos;
- notebooks de Machine Learning;
- discussão crítica (engenharia + computação).

> O foco é **integração e rastreabilidade**, não automação de dimensionamento.

## Componentes
- `shared_parameters/`: parâmetros compartilhados (ABNT + rastreabilidade acadêmica).
- `modelagem/`: guias curtos de modelagem (rasas e profundas).
- `export/`: esquema de exportação (CSV) para uso nos notebooks.

## Template Revit
A disciplina pode utilizar um **template-base** já existente (arquitetura/folhas/cortes).
Os alunos são incentivados a aprimorar o modelo, desde que mantenham os parâmetros CV721A.
